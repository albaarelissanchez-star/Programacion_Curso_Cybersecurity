# Proyecto Gestor de Contraseñas Seguras

Programa en PSeInt que permite registrar usuarios y contraseñas, verificar su nivel de seguridad y mostrar alertas cuando alguna es débil.

## Descripción
El algoritmo guarda los usuarios en un vector junto con sus contraseñas, 
evalúa cada una mediante una función que analiza su fortaleza y al final muestra cuántas son seguras y cuántas no.

## Funcionalidades
- Registrar usuarios y contraseñas.  
- Verificar la fuerza de cada contraseña.  
- Mostrar alertas y un resumen final.

## Ejecución
Abrir el archivo en PSeInt, ejecutar el programa e ingresar los datos que solicita.
